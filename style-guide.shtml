stringer

Stringer is a robust, responsive framework with plenty of optional building blocks. It's not a feature-complete boilerplate, but a good starting point upon which you can develop custom themes.

Please note: this project is a sandbox. It changes often. It might have a bug or two, from time-to-time. I try to push only bug-free versions, but I make no promises, and I offer no support of any kind. It's yours to use as you wish.

A stringer is a wooden strip that runs the length of a surfboard, designed to give the board strength & rigidity.

  • See the repo for more info
  • I use Codekit to manage this project. My config file is in the repo, if you're a Codekit user.
  • I'm using the (very customizable) jQuery mmenu system for mobile navigation.
  • Stringer uses an SHTML file structure for SSI (server side includes), as per the practice at my agency. With very little effort, you can adapt it to use PHP, for instance.
  • My build theory: Core SASS partials lay the groundwork, with one _theme.scss file with lots of BEM styles to build up your design (so the core framework remains unchanged). You can follow this path, or get into the partials and customize them to your heart's content.

Images

By default, (regular, non-picturefill-enabled) images appear at their native size. They re-size relative to their container on smaller screens.

 

Images with class="fit" will always fill the container, no matter their native size.
